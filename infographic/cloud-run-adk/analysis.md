# Infographic analysis

- **Topic:** Long-running, security-sensitive repository repair with Gemini 3.5 Flash, Google ADK, and native Cloud Run sandboxes.
- **Audience:** Cloud platform engineers, agent framework developers, security engineers, and developer-tool builders.
- **Learning objective:** Show the complete execution lifecycle and the evidence proving why the sandbox is needed.
- **Data shape:** Seven ordered stages plus one cross-cutting security-boundary note.
- **Selected layout:** `linear-progression` with a compact evidence dashboard.
- **Selected style:** `technical-schematic` adapted to Google Cloud colors.
- **Aspect:** 16:9 landscape, 1600×900.
- **Language:** English.
- **Typography:** Google Sans family.
- **Integrity constraint:** Every number is copied from `assets/evidence/full-review.json`; no synthetic values.
- **Rendering decision:** Deterministic HTML/SVG screenshot rather than generative text rendering, to preserve exact labels, hashes, units, and test counts.
